Feilds of programing languages :-

The field of web development: It means the development and programming of web applications, and this simply means the design and programming of websites. Hyper Text Markup Language, also known as html, is a descriptive language used to build and design web pages, then comes css, which is the language responsible for coordinating colors and shapes in different web pages, and after that comes the dynamic language And interaction with the user is the Javascript language, and after studying these three languages, you have somehow mastered the design of web pages. Then comes the most important part in this field, which is the language used in the Back-End, meaning the language that deals with databases on the site, and here you can study Python or PHP, only one of them is enough, in addition to studying databases, of course SQL and there Many of them you can choose only one will do the job. After that, you must study some libraries for the web domain such as Bootstrap, as well as the Jquey JavaScript library, and some other frameworks such as the framework for the Python language, which is Flask or the Django framework, one of these frameworks is sufficient. After that, you must learn the necessary techniques that each developer needs, which are the various educational platforms, forums, freelance sites, and communities of the programming language that you use, and you must study the Git version management program and the Github platform because it is the main gateway to the projects you undertake. Desktop applications field: This field requires you to study one of the following languages: java, C#, Pyhton, VB.NET. In addition, you must study databases such as MySql, SQL Server, Sqlite3 and others. The field of mobile applications: This field is very popular at the present time, and entry to it requires learning one of the following languages: Java with Android Studio (development environment), C# with Xamarin environment, and Kotlin with Android Studio environment and of course you must learn databases as mentioned before.
